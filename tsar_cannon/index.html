Tsar Cannon

This is a setup that exists. Guide source: John Beak's page.
Tsar Cannon is yet another TD 8L opener thing. It has 83.33% buildrate (93.89% with mirror), and a single build for bag2 (with 180). It is currently the highest average scoring known 8L TD, with 98.69% PC chance.



Tsar Cannon is buildable when J_tetramino>S_tetramino and I_tetramino>O_tetramino.
