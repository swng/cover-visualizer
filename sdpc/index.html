SDPC

This is a setup that exists. Some other resources: this sheet; the four.lol page; the JP Wiki page.

SDPC is TSS - TSD - 8L PC opener. It has various alt paths such as SDPCspin, extended SDPC, mechTSD.



Bag 1 TSS is buildable if O_tetramino precedes Z_tetramino. This single dependency gives it 66.67% coverage or 75% coverage inc. mirror. The orientation with TSS on left has slightly better kpp.


See which queues apply to which setups.

The 2nd bag can always be stacked the as per the main path, though a unique L_tetramino spin is required for some bags.
There are also some alt paths for bag 2:
An alt O_tetramino placement (if bag allows 66.67% of the time) gives a better PC field.
Alt L_tetraminoJ_tetramino placements build SDPCspin.
Swapping O_tetramino and L_tetramino placements builds extended SDPC.

If going for PC path, complete an 8L PC on bag 3. Note that coverage on this page is computed assuming tetrio 180 kicks.
If doing SDPCspin, build a c spin. There are two builds, depending on Z_tetraminoJ_tetraminoL_tetramino order.
If building extended SDPC: stickmancomic's hamburger can always be built with an LJ congruency. For a bigger spike, note that the starting field is essentially a flat 6-3 quad well with an O_tetramino chunk moved. Thus, most openers with O_tetramino to the side can usually be built on top of this well. This can produce the opener's damage + extra quad spike.
