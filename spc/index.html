SPC

This is a page of bruteforce-generated SPCs with 98%+ PC chance and good average scoring. Data is calculated with tetrio 180s kicktable.
For parity reasons, T_tetramino SPC fully outscores T_tetramino DPC. For the rest, their sets outscore their DPC counterparts on average but don't fully replace. Optimally, one would fully use T_tetramino SPC, and a mix of SPC and DPC for the rest.
DPC is a setup for a T-Spin Double ▸ Perfect Clear. It requires an empty field after eight cleared lines. With the setup using one leftover piece + one bag for the setup.
SPC is a setup for a T-Spin Single ▸ Perfect Clear. It requires an empty field after eight cleared lines. With the setup using one leftover piece + one bag for the setup.
